\section{Motivation}

\begin{frame}[t]{Calcium imaging: recording neuronal activity}
\begin{columns}[c]
% ---- Left: text ----------------------------------------------------------
\begin{column}{0.53\textwidth}
\begin{itemize}
\item Calcium imaging lets us see brain activity directly: when a neuron fires,
      calcium rises and a fluorescent indicator brightens, so active neurons
      \enquote{light up} over time (a 2D${}+{}$time movie).
      \cite{grienberger2012calcium}
\item A key window into how the brain works --- which neurons are active, and
      when --- used across many species and labs.
\item But the raw movie is not yet science: every neuron must first be located
      (segmented) before its activity can be measured.
\item Doing this by hand does not scale --- manual annotation is slow,
      subjective, and inconsistent between annotators.
\item So the value of this project is an automatic segmentation tool that makes
      calcium-imaging analysis fast, objective, and scalable.
\end{itemize}
\end{column}
